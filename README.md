# PDF 思维导图摘录插件

一个强大的思源笔记插件，用于从PDF文档中提取文本和图片，并以思维导图形式组织和展示标注内容。

## 功能特性

### 📚 项目管理
- 支持创建多个PDF项目，每个项目可包含多本PDF
- 统一管理所有PDF文档和标注内容
- 项目级统计信息（PDF数量、标注数量等）

### 📄 PDF处理
- 支持多PDF文档同时管理
- PDF页面浏览和导航
- 实时PDF内容预览

### ✍️ 标注功能
- **文字摘录**：选择PDF中的文字内容进行标注
- **图片摘录**：圈选PDF中的图片进行摘录
- **多级标注**：支持标题、段落、句子等多个层级
- **实时同步**：标注自动同步到思源笔记对应文档

### 🎯 视图模式
- **分屏模式**：PDF预览与标注列表并排显示
- **列表模式**：专注标注内容管理
- **思维导图模式**：可视化展示标注间的层级关系

### 🔄 数据同步
- 标注内容实时保存到思源笔记
- 支持标注的增删改查
- 自动维护标注与PDF页面的关联

## 安装方法

1. 将插件文件夹复制到思源笔记的 `plugins` 目录下
2. 重启思源笔记
3. 在设置中启用插件

## 使用指南

### 创建项目
1. 点击"项目"按钮打开项目管理面板
2. 点击"新建项目"创建新项目
3. 选择第一个PDF文件导入

### 添加PDF
1. 在项目管理面板中选择目标项目
2. 点击"添加PDF"按钮
3. 选择要添加的PDF文件

### 文字摘录
1. 在PDF预览区选择要摘录的文字
2. 点击"创建标注"按钮
3. 选择合适的标注级别（标题/段落/句子）

### 图片摘录
1. 在PDF预览区圈选要摘录的图片区域
2. 系统会自动创建图片标注
3. 图片会保存到思源的资源库

### 管理标注
- 点击标注列表中的项目查看详细内容
- 可编辑、删除或移动标注
- 支持标注的合并与拆分

## 技术架构

### 前端技术栈
- Vue 3 + TypeScript
- Vite 构建工具
- PDF.js 用于PDF渲染
- D3.js 用于思维导图可视化

### 核心组件
- `PDFViewer`：PDF渲染和交互
- `AnnotationList`：标注列表管理
- `MindMapViewer`：思维导图可视化
- `AnnotationEditor`：标注编辑界面

### 数据管理
- 本地存储项目和标注元数据
- 与思源API集成进行文档同步
- 支持增量更新和冲突解决
