# fangmu-pdf摘录助手

一个 PDF 标注摘录与思维导图生成工具，帮助你高效地从 PDF 文档中提取知识并组织成结构化笔记。
(注意: 思维导图功能暂未实现)

## ✨ 核心功能

### 📚 多 PDF 项目管理
- 创建项目，将多本 PDF 统一管理
- 支持项目内多 PDF 切换浏览
- 项目数据持久化存储

### 📝 文字摘录
- 在 PDF 中选择文本，一键创建标注
- 支持多种标注级别：文档标题、H1-H5 标题、正文标注
- 标注自动同步保存到思源文档
- PDF 中实时显示高亮标记

### 📷 图片摘录
- 框选 PDF 区域，截取为图片
- 图片自动上传到思源资源目录
- 支持添加标注级别和笔记
- PDF 中显示 📷 角标标识图片摘录

### 🎨 标注管理
- 标注列表实时显示所有摘录内容
- 支持按页码排序浏览
- 点击标注可快速定位到 PDF 对应位置
- 支持编辑标注内容、添加笔记
- **拖拽合并**：将标注拖拽到标题下方进行合并整理

### 🗑️ 多种删除方式
- **PDF 内删除**：
  - 悬停删除：鼠标悬停在高亮标记上，右上角显示红色 × 按钮，点击删除
  - 键盘删除：点击高亮标记选中（边框变白、背景加深），按 Delete 或 Backspace 键删除
- **列表内删除**：在标注列表中点击删除按钮
- **双向同步**：无论从 PDF 还是列表删除，都会同步移除思源中的对应块

### ⌨️ 快捷操作
- **点击标注**：选中标注（高亮显示）
- **Delete/Backspace 键**：删除选中的标注
- **双击标注**：跳转到 PDF 对应位置

## 🚀 使用方法

### 1. 创建项目
点击顶部「+ 新项目」按钮，输入项目名称并选择一个 PDF 文件。

### 2. 添加 PDF
在项目管理中可以继续添加更多 PDF 到当前项目。

### 3. 摘录内容
- **文字模式**：选择工具栏「文字」按钮，在 PDF 中拖选文本
- **图片模式**：选择工具栏「图片」按钮，在 PDF 中框选区域

### 4. 设置级别
在摘录前，可以在工具栏选择标注级别：
- 文档标题：将作为目标文档的标题
- H1-H5：生成对应级别的标题
- 正文标注：普通文本高亮

### 5. 指定目标文档（可选）
在「目标」输入框中搜索并选择思源文档，标注将保存到该文档。

### 6. 管理标注
- **查看**：标注列表实时显示所有摘录
- **定位**：双击标注项跳转到 PDF 对应位置
- **编辑**：点击编辑按钮修改标注内容和笔记
- **合并**：拖拽标注到标题下方进行合并整理
- **删除**：
  - 在 PDF 中悬停点击 × 按钮删除
  - 在 PDF 中点击选中后按 Delete 键删除
  - 在列表中点击删除按钮删除

## 📖 使用场景

### 学术研究
- 阅读论文时快速摘录关键观点
- 按标题级别组织文献结构
- 导出 Markdown 格式笔记

### 知识整理
- 多本教材统一管理
- 重要概念分类标注
- 图片截图保存图表

### 读书笔记
- 划线摘录精彩段落
- 添加个人思考和笔记
- 生成结构化读书笔记

## 🎯 特色亮点

1. **无侵入式设计**：标注数据存储在思源文档中，不依赖插件也能查看
2. **多级标题支持**：直接生成 H1-H5 标题，方便构建文档结构
3. **双向联动**：PDF 与标注列表实时联动，快速定位
4. **图片摘录**：支持截图保存 PDF 中的图表、公式等
5. **项目化管理**：多个 PDF 文件统一管理，适合专题研究
6. **便捷删除**：支持 PDF 内直接删除标注，双向同步

## ⚙️ 技术实现

- 基于 PDF.js 渲染 PDF 文档
- 使用 Vue 3 + TypeScript 开发
- 数据存储在思源文档块属性中
- 支持思源主题样式适配
- DOM 高亮层实现标注交互

## 📌 注意事项

1. 首次使用需要先创建项目并导入 PDF
2. 标注保存需要有效的目标文档（未指定时会尝试保存到最近编辑的文档）
3. 图片摘录会生成 PNG 文件存储在 assets 目录
4. 建议定期通过项目管理检查项目状态
5. 删除操作会同步移除思源中的对应块，请谨慎操作
