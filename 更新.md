# 插件更新日志

## 2026/2/27 更新

### 新增功能

#### 1. 学习集管理增强 - PDF 选择器

**文件**: `src/components/LearningSetManager.vue`

**功能描述**:
- 在给学习集添加 PDF 时，支持从思源工作空间 `/data/assets/` 目录中已上传的 PDF 文件中选择
- 提供下拉框选择器，显示所有可用的 PDF 文件
- 保留手动输入路径功能，并添加自动补全建议
- 支持刷新 PDF 列表

**UI 特性**:
- PDF 下拉选择框带刷新按钮
- 分隔线区分两种输入方式（选择 vs 手动输入）
- 加载状态显示
- 空状态提示（当没有 PDF 时显示引导信息）

**使用方法**:
1. 打开学习集详情面板
2. 点击"添加 PDF"按钮
3. 在下拉框中选择已上传的 PDF，或手动输入路径
4. 点击"添加"按钮

## 开发注意事项

### 1. 思源 API 使用

```typescript
import { readDir, type FileInfo } from '../api/siyuanApi';

// 读取目录
const list = await readDir('/data/assets');

// 过滤 PDF 文件
const pdfFiles = list
  .filter((f: FileInfo) => !f.isDir && f.name.toLowerCase().endsWith('.pdf'))
  .map((f: FileInfo) => ({
    path: `/data/assets/${f.name}`,
    name: f.name
  }));
```

### 2. 学习集服务使用

```typescript
import { LearningSetService } from '../services/learningSetService';

// 获取所有学习集
const sets = LearningSetService.getAllLearningSets();

// 添加 PDF 到学习集
LearningSetService.addPdfToLearningSet(setId, pdfPath);

// 移除 PDF
LearningSetService.removePdfFromLearningSet(setId, pdfPath);
```

### 3. 组件通信模式

```typescript
// 定义 emits
const emit = defineEmits<{
  'set-selected': [set: LearningSet];
  'pdf-open-request': [pdfPath: string];
  'annotation-focus-request': [annotation: PDFAnnotation];
  'page-jump-request': [pdfPath: string, page: number];
}>();

// 触发事件
emit('pdf-open-request', pdfPath);
```

## 技术栈

- **前端框架**: Vue 3 + TypeScript
- **构建工具**: Vite
- **PDF 渲染**: pdfjs-dist
- **思维导图**: markmap-lib, markmap-view
- **数据存储**: 思源笔记块存储 API
- **算法**: SM-2、FSRS 间隔重复算法
